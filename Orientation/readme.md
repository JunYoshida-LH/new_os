Tips and Deeper side of New OS
-----------------------------------------------------------------

OS manualには、より高速に、より効率的に作業する為の情報が含まれています。

小さなティップス群と大きな原理的ルール群などがあります。
他の役割がどのように動いているか、OSがどのように動いているのか、どのように雇用を進めているのかといった情報等、特有の方法が記載されています。
